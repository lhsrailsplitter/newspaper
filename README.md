# The Railsplitter

Student newspaper of Abraham Lincoln High School — Los Angeles, CA.

Plain HTML/CSS/JS — no framework, no build step. Open any `.html` file in a browser and it works. All content lives in `content.json`.

## Editor

Go to `editor.html`, sign in with your GitHub personal access token, and you're in. Changes push directly to this repo as commits.

You need to be in the `lhsrailsplitter` org first — ask your advisor.

## Running locally

```bash
python3 -m http.server
```
